# Ter Project - AI Guidance

你目前正在协助开发 Ter (SSH 可视化平台)。

## 核心任务指令
当你执行 UI 或逻辑修改后，**必须** 遵循以下视觉验证流程：

1.  **保存代码**：修改 `src/App.vue` 或后端。
2.  **触发快照**：执行 `curl http://127.0.0.1:1414`。
3.  **视觉验收**：检查 `snap/` 下最新的图片。

**[强制性规则]**:
当你启动 Vite 服务时，**必须** 带有 `--host 0.0.0.0` 参数，并且 **必须** 在 `vite.config.ts` 中加入 `server: { allowedHosts: true }` 以绕过所有域名限制。

## 插件注入规则
*   **通知注入**：返回 `{"type": "text", "message": "..."}`。
*   **图表注入**：返回 `{"type": "chart", "message": [数据数组]}`。

*   控制 API：`http://127.0.0.1:1414` (内部触发截图)
